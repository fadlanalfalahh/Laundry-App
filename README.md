# Laundry App

**Laundry App** adalah aplikasi manajemen laundry berbasis Flutter yang digunakan oleh kios *Laundry* untuk mengelola layanan, transaksi, dan laporan pendapatan.  
Aplikasi ini dibuat khusus untuk kebutuhan operasional laundry, dengan sistem API backend (PHP & MySQL) agar data transaksi tersimpan.

---

## Fitur Utama
- **Login Multi-Role** — Sistem autentikasi untuk *admin* dan *kasir*.
- **Manajemen Paket Laundry** — CRUD paket laundry berdasarkan berat (kg) dan satuan (item), dengan harga.  
- **Transaksi Laundry** — Input pesanan, perhitungan total harga, dan pencatatan status pengerjaan.  
- **Cetak Struk Transaksi** — Struk digital yang dapat dicetak atau disimpan.  
- **Laporan Transaksi** — Rekap data pendapatan harian, mingguan, dan bulanan.  

---

## Teknologi yang Digunakan
- **Frontend (Mobile):** Flutter (Dart)  
- **Backend (API):** PHP Native
- **Database:** MariaDB  
- **PDF Generator:** `pdf` package Flutter  

---

## Tampilan Aplikasi
| Halaman | Deskripsi |
|----------|------------|
| Login | Autentikasi untuk admin dan kasir |
| Paket Laundry | CRUD layanan berdasarkan berat dan satuan |
| Transaksi | Input pesanan dan perhitungan total harga |
| Struk | Cetak atau simpan bukti transaksi |
| Laporan | Rekap pendapatan berdasarkan tanggal |
